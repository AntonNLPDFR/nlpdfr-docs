## Wat is NLPDFR nou precies?

**NLPDFR** is een Nederlandstalige **FiveM** community. Onze focus ligt op de **FivePD** PvE ervaring. Daarnaast bieden we ook regelmatig een **georganiseerde surveillance** aan.





> Informatie vertoont op deze website is doelgericht gemaakt voor de NLPDFR community. Of de informatie hier vertoont werkt op andere servers, kunnen wij niet garanderen.
